# WIREFRAME.md

# 화면 와이어프레임 명세서 (UI Wireframes)

본 문서는 플랫폼의 핵심 화면 4가지에 대한 구조를 텍스트 및 그리드 기반 모형(ASCII Wireframes)으로 가시화한다.

---

## 1. 홈 포털 대시보드 와이어프레임 (Home Portal Layout)

* **경로**: `/` (기본 대시보드 메인)
* **목적**: 활성 워크스페이스 상태 요약, Quota 실시간 추적 및 퀵 액션 진입.

```text
+-------------------------------------------------------------------------------------------------+
|  [Logo] Smart Dashboard   | Workspace: [ My Store A (Enterprise) v ]      [Alert(3)]  [Profile] |
+-------------------------------------------------------------------------------------------------+
|  [Sidebar]    |                                                                                 |
|  - 홈         |  [ 환영합니다! 김민수 셀러님 ]                                                  |
|  - 상품 기획  |  오늘도 고객의 숨겨진 결핍(JTBD)을 발견하여 혁신적인 상품을 기획해 보세요.      |
|  - 감사 센터  |                                                                                 |
|  - 탄력 모니터|  +---------------------------------+  +--------------------------------------+  |
|  - 설정       |  |  실시간 API Quota 사용 현황      |  |  신규 분석 퀵 온보딩                 |  |
|               |  |  [||||||||||||||||||||||      ]  |  |  네이버 쇼핑 상품 키워드를 입력해    |  |
|               |  |  85% 사용 (85 / 100회 호출)     |  |  즉시 AI 기획을 시작해 보세요.       |  |
|               |  |  * 이번 달 15회 잔여            |  |  [ 시장성 분석 시작하기 ]            |  |
|               |  +---------------------------------+  +--------------------------------------+  |
|               |                                                                                 |
|               |  +---------------------------------------------------------------------------+  |
|               |  |  최근 운영 알림 및 이벤트 이력 (Recent Audits)                             |  |
|               |  |  - [2026-06-28 19:45] [INFO] User login successfully from IP 1.2.3.4       |  |
|               |  |  - [2026-06-28 19:20] [WARN] Quota limit is approaching 80% threshold      |  |
|               |  |  - [2026-06-28 18:02] [INFO] AI Strategy approved for Product #12         |  |
|               |  +---------------------------------------------------------------------------+  |
|               |                                                                                 |
+-------------------------------------------------------------------------------------------------+
```

## 2. AI 상품 기획 스토리보드 와이어프레임 (AI Strategy Storyboard)

* **경로**: `/products/[productId]/strategy`
* **목적**: AI가 도출한 8단계 상세페이지 기획안 전개 및 시안 확인, 대표 승인.

```text
+-------------------------------------------------------------------------------------------------+
|  [Logo] Smart Dashboard   | Workspace: [ My Store A (Enterprise) v ]      [Alert(3)]  [Profile] |
+-------------------------------------------------------------------------------------------------+
|  [Sidebar]    |  < 상품 상세: 친환경 텀블러 (#12)                                               |
|  - 홈         |  +---------------------------------------------------------------------------+  |
|  - 상품 기획  |  |  [ 분석 리포트 ]  [ * AI 기획 스토리보드 ]  [ 판매 성과 환류 ]            |  |
|  - 감사 센터  |  +---------------------------------------------------------------------------+  |
|  - 탄력 모니터|                                                                                 |
|  - 설정       |  상태: [ 승인 대기중 ] (최종 기획안 검토 후 승인을 완료해 주세요)               |
|               |                                                                                 |
|               |  +---------------------------------------------------------------------------+  |
|               |  |  1단계: 문제 정의 및 호기심 유발 (Attention)  [ v ] 아코디언 토글 확장     |  |
|               |  |  - 카피 문구: "텀블러를 매일 세척하기 귀찮으셨죠? 이제 솔을 버리세요."       |  |
|               |  |  - 컨셉 프롬프트: 'A close up photography of dirty tumbler...'              |  |
|               |  +---------------------------------------------------------------------------+  |
|               |  |  2단계: 소비자 결핍 제시 (Agitation)          [ > ] 닫힌 상태             |  |
|               |  +---------------------------------------------------------------------------+  |
|               |  |  3단계: 핵심 USP 솔루션 제안 (Solution)        [ > ] 닫힌 상태             |  |
|               |  +---------------------------------------------------------------------------+  |
|               |                                                                                 |
|               |  +---------------------------------------------------------------------------+  |
|               |  |  생성된 이미지 썸네일 시안 (FLUX AI Generated)                              |  |
|               |  |  [ Thumbnail 1 ]   [ Thumbnail 2 ]   [ Thumbnail 3 ]   [ Thumbnail 4 ]    |  |
|               |  +---------------------------------------------------------------------------+  |
|               |                                                                                 |
|               |                       [기획안 반려]       [ 기획안 최종 승인 (Approve) ]         |
+-------------------------------------------------------------------------------------------------+
```

## 3. Sourcing & Margin 계산기 와이어프레임 (Sourcing & Margin Calculator)

* **경로**: `/products/[productId]/sourcing`
* **목적**: 상품 마진 및 소싱 원가, 이익 구조 시뮬레이션.

```text
+-------------------------------------------------------------------------------------------------+
|  [Logo] Smart Dashboard   | Workspace: [ My Store A (Enterprise) v ]      [Alert(3)]  [Profile] |
+-------------------------------------------------------------------------------------------------+
|  [Sidebar]    |  < 소싱 및 마진 계산기 시뮬레이터                                               |
|  - 홈         |                                                                                 |
|  - 상품 기획  |  +---------------------------------+  +--------------------------------------+  |
|  - 감사 센터  |  |  비용 입력 필드 (Input Variables) |  |  실시간 수익성 요약 (Live Summary)   |  |
|  - 탄력 모니터|  |                                 |  |                                      |  |
|  - 설정       |  |  1. 예상 판매가: [ 25,000 ] 원  |  |  - 소싱 총 원가: 9,200 원            |  |
|               |  |  2. 사입 원가:   [  6,500 ] 원  |  |  - 수수료 (네이버): 1,450 원          |  |
|               |  |  3. 광고비 비율: [   15 ] %     |  |  - 마진 금액: 14,350 원              |  |
|               |  |  4. 배송비 부담: [  3,000 ] 원  |  |  - 최종 마진율: [ 57.4% ] (우수)     |  |
|               |  +---------------------------------+  +--------------------------------------+  |
|               |                                                                                 |
|               |                      [ 초기화 ]             [ 마진 시뮬레이션 저장 ]             |
+-------------------------------------------------------------------------------------------------+
```
